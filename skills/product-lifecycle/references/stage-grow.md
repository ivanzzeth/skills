# Stage 6: GROW — Acquire & Retain Users

## Goal

Drive awareness, acquire first users, and build retention loops.

## Workflow

### Step 1: SEO foundation

Invoke `/seo-audit` on the product's landing page or docs site.

Fix critical SEO issues:
- Title tags, meta descriptions
- Heading hierarchy
- Page speed
- Mobile responsiveness
- Schema markup

### Step 2: Landing page

Invoke `/landing-page-design` to create or improve the landing page.

Must include:
- Clear value proposition (above fold)
- Problem → Solution narrative
- Social proof (if available)
- Single clear CTA
- SEO optimized

### Step 3: Marketing copy

Invoke `/copywriting` to create:
- Homepage copy
- Product descriptions
- Feature explanations
- CTA copy variants

Reference the product marketing context (`.agents/product-marketing-context.md`)
created in Stage 2.

### Step 4: Email marketing

Invoke `/email-sequence` to design:
- Welcome sequence (3-5 emails)
- Onboarding sequence
- Re-engagement sequence

### Step 5: Lead generation

Invoke `/lead-research-assistant` to identify:
- Target communities and forums
- Potential early adopters
- Partnership opportunities
- Content distribution channels

### Step 6: Financial model

Invoke `/startup-financial-modeling` to project:
- Unit economics (CAC, LTV, payback period)
- Revenue projections (month 1-12)
- Break-even analysis

Document everything in `06-growth-plan.md`.

## Ongoing Growth Loop

After initial setup, the growth stage is ongoing. Key activities:
- Weekly: Check analytics, iterate on copy/SEO
- Monthly: Review financial model vs actuals
- Quarterly: Re-evaluate channels, add/drop based on ROI

## No Gate

GROW is the final stage. There is no gate to pass.
The product is live — iterate, measure, and improve continuously.
